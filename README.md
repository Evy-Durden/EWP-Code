# Misconfiguration Detection Tool

This Python-based CLI tool is used for detecting misconfigration on Linux machines. The focus lies on privilege escalation.
It has eBPF integration to analyze the system not just statically but also behaviorally. It scans for misconfigurations and acts as a runtime security analyzer.
